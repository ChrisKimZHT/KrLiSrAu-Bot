# codeforces

查询 Codeforces 比赛

### 功能用法

触发指令：`codeforces` / `cf`

```
codeforces <数量> - 查询最近的指定数量的比赛
```

#### 查询比赛

查询指定数目的比赛，默认为 3 个。

**示例：**`cf 2`

**结果：**

```
【Codeforces比赛】
Codeforces Round 870 (Div. 2)
Type: CF
Duration: 120 (min)
Time: 2023/05/04(Thu) 22:35
-------------------------
Codeforces Round (Div. 1)
Type: CF
Duration: 120 (min)
Time: 2023/05/08(Mon) 20:05
-------------------------
```
