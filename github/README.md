# github

查询指定用户 GitHub 贡献图

### 功能用法

触发指令：`ghcontribute` / `ghc`

```
ghcontribute <用户名> - 查询指定用户的GitHub贡献图
```

#### 查询贡献图

查询指定用户的 GitHub 贡献图，基于 https://ghchart.rshah.org/

**示例：**`ghc ChrisKimZHT`

**结果：**

![](https://assets.zouht.com/img/md/KrLiSrAu-Bot-README-01.png)
