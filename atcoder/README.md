# atcoder

查询 AtCoder 比赛信息

### 功能用法

触发指令：`atcoder` / `atc`

```
atcoder [选项]
    空 - 显示所有类别
    abc - 显示Beginner类别
    arc - 显示Regular类别
    agc - 显示Grand类别
    ahc - 显示Heuristic类别
```

#### 查询比赛

**示例：**`atcoder`

**结果：**

```
【Atcoder比赛】
Start Time  | Name  | Dura.
05/13(Sat) 20 | ABC301 | 01:40
05/14(Sun) 20 | ARC160 | 02:00
05/21(Sun) 20 | AGC062 | 03:00
05/27(Sat) 20 | ABC303 | 01:40
05/28(Sun) 20 | ARC161 | 02:00
04/23(Tue) 14 | AHC020 | 04:00
```
